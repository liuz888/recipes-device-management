## 准备工作
1. 在小程序项目目录中执行命令安装 npm 包：
```
npm install
```
2. 点击开发者工具中的菜单栏：工具 --> 构建 npm

## 启动开发

调试请使用演示企业和用户账号
  - 获取账号，请前往 https://developer-pre.rootcloudapp.com/recipes/device-management/
